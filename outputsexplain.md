# Output Explanation

This document explains the structure and contents of the `outputs/` folder generated by the weather analysis pipeline.

## 1. `annual_stats/`

Contains annual summaries for each location and variable.

- **`all_annual_stats.RDS` / `.csv`**: Combined dataset of annual statistics.
- **`[Location]_annual_stats.csv`**: Annual statistics for specific locations.
- **Columns**:
  - `Mean`: Average value for the year.
  - `Total`: Sum of values (relevant for Precipitation).
  - `Max` / `Min`: Maximum and Minimum monthly values.
  - `SD`: Standard Deviation (Variability within the year).
  - `CV`: Coefficient of Variation (SD / Mean \* 100).

## 2. `seasonal_stats/`

Contains seasonal summaries (Northeast Monsoon, Southwest Monsoon, etc.).

- **`all_seasonal_stats.RDS` / `.csv`**: Combined dataset of seasonal statistics.
- **`[Location]_seasonal_stats.csv`**: Seasonal statistics for specific locations.
- **Columns**: Similar to annual stats, but aggregated by Season.

## 3. `period_stats/`

Contains statistics aggregated over specific time periods (e.g., 1870-2025, 1991-2025).

- **`all_period_stats.csv`**: Summary of Mean, SD, Min, Max, and CV for each Period, Location, Variable, and Season.

## 4. `anomalies/`

Contains calculated anomalies relative to the baseline (1961-1990).

- **`all_annual_anomalies.RDS` / `.csv`**: Annual anomalies.
- **`all_seasonal_anomalies.RDS` / `.csv`**: Seasonal anomalies.
- **`period_total_anomalies.csv`**: Sum of anomalies for each defined time period.
- **Columns**:
  - `Anomaly`: The difference between the observed value and the baseline mean.
  - `Standardized_Anomaly`: Anomaly divided by the baseline standard deviation.

## 5. `trends/`

Contains results of the Mann-Kendall Trend Test and Sen's Slope Estimator.

- **`all_trends.RDS`**: Combined trend results.
- **`[Location]_trends.csv`**: Trend results for specific locations.
- **Columns**:
  - `MK_tau`: Mann-Kendall Tau statistic.
  - `MK_Z`: Mann-Kendall Z-statistic (Standard normal deviate).
  - `MK_p_value`: P-value indicating significance of the trend.
  - `Sens_Slope`: Magnitude of the trend (change per year).
  - `Significance`: Stars indicating significance level (\* p<0.1, ** p<0.05, \*** p<0.01).

## 6. `correlation/`

Contains spatial correlation analysis (Location vs. Location) for both Rainfall and Temperature, analyzed over specific time periods.

- **`[Period]/` Subfolders** (e.g., `1870-2025/`, `1870-1900/`):
  - **`correlation_precipitation.csv`**: Correlation matrix for Precipitation.
  - **`correlation_temperature.csv`**: Correlation matrix for Temperature.
  - **`heatmap_precipitation.png`**: Enhanced heatmap visualization for Precipitation correlation.
  - **`heatmap_temperature.png`**: Enhanced heatmap visualization for Temperature correlation.
  - **`correlation_summary.csv`**: Summary of the most correlated district for each location.

## 7. `plots/`

Contains time-series visualizations.

- **Structure**: `[Period] / [Location] / [Filename].png`
- **Content**: Plots of Annual and Seasonal data for Precipitation and Temperature.
- **Features**:
  - **Trend Line**: Dashed black line showing the linear trend.
  - **Annotations**: Text box showing:
    - `MK Z`: Mann-Kendall Z-value.
    - `MK Tau`: Mann-Kendall Tau statistic.
    - `MK p-value`: Significance p-value.
    - `Sen's Slope`: Magnitude of the trend.
  - **X-Axis**:
    - For `1870-2025`: 5-year intervals.
    - For other periods: 2-year intervals.

## 8. `advanced_analysis/`

Contains results from Wavelet, Fourier, and Change-point analysis.

- **`advanced_analysis_summary.csv`**: Summary of detected change points and dominant periodicities.
- **`[Period] / [Location] /`**:
  - **`wavelet_... .png`**: Wavelet power spectrum plots showing periodicities over time.
